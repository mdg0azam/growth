Plan Logic

  • Daily anchors: Quran, hadith, exercise, healthy eating.
  • Weekly anchors: 3 seerah sessions, 4 Arabic sessions, 2 calligraphy sessions, 3 character-growth sessions.
  • Professional blocks: GIS/RS foundations → climate/research → data science/SQL → JavaScript/Web GIS.
  • Review rhythm: 30–45 minutes every week to audit carry-over tasks and refine the next week.

Front-End Design Notes

  • Semantic HTML sections, cards, and accessible labels.
  • Responsive CSS grid layout for desktop and mobile.
  • Modern visual system with glassmorphism panels, progress rings, and clean spacing.
  • Vanilla JavaScript interactions only — no external libraries, fully portable as one downloadable file.

Execution Rules

  • Missed a task? Move it into the next review block instead of abandoning the system.
  • For reading-heavy weeks, aim for consistency over intensity.
  • Keep one notes system only: a single notebook or digital doc for reflections, vocabulary, and technical summaries.
  • At every milestone (W4/W8/W12/W16/W20/W24), produce one short output artifact.
